# ⚡ Sports Analytics Dashboard — Fatigue & Injury Tracker

A real-time, AI-powered sports analytics dashboard that tracks player fatigue levels, injury risk, stamina, and performance stats. Built with React and Vite, featuring a premium dark UI with neon accents, animated charts, and intelligent recommendations.

![React](https://img.shields.io/badge/React-18-61DAFB?logo=react&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-5-646CFF?logo=vite&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?logo=javascript&logoColor=black)

## ✨ Features

- **Live Player Monitoring** — Track fatigue index, injury risk, stamina, and match count in real-time.
- **Interactive Player Selector** — Click any player card to instantly view their detailed metrics.
- **Animated Performance Bars** — Speed, Strength, Endurance, and Recovery stats with smooth fill animations.
- **Fatigue Trend Charts** — Mini SVG sparkline charts showing fatigue progression over the last 8 matches.
- **AI Recommendations** — Automated fitness recommendations (REST / MONITOR / FIT) based on player data.
- **Premium Dark UI** — Neon glow effects, 3D perspective rotations, scan-line animations, and stadium lighting.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/YOUR-USERNAME/sports-analytics-dashboard.git
   ```

2. **Navigate into the project**
   ```bash
   cd sports-analytics-dashboard
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the dev server**
   ```bash
   npm run dev
   ```

5. Open [http://localhost:5173](http://localhost:5173) in your browser.
